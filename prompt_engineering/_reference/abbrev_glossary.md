# Brooks 缩写术语表（精选，参考层）

来源：`阿布缩写翻译及解释.txt`。程序路由以 `pattern_enum.md` 为准。

## 结构与形态

| 缩写 | 英文 | 中文要点 |
|------|------|----------|
| FF | Final Flag | 趋势末端水平旗形，突破常失败反转 |
| 尖峰旗形 | Spike Flag | 尖峰中第一次整理；**允许** SPS 顺向突破入场（≠ FF） |
| MW | Micro Wedge | 3–4 棒楔形 |
| NW | Nested Wedge | 嵌套楔形 |
| MM | Measured Move | 测量移动目标 |
| MTR | Major Trend Reversal | 主要趋势反转 |
| MRV | Minor Trend Reversal | 小反转，多半回撤/区间 |
| OB | Outside Bar | 外包棒 |
| IB | Inside Bar | 内包棒 |
| OD | Outside Down | 外包向下 |
| 2BR | Two-Bar Reversal | 双棒反转 |
