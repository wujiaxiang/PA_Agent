# 逐棒分析检查单（Al Brooks）

本文件为**阶段二** playbook：口诀与特殊结构速查。阶段一仅用 `市场诊断框架` 第三层 + `bar_by_bar_summary`（K5–K1 摘要，恰好 5 条）+ **文件16** 字典；**不在阶段一加载本文件**。

核心原则：先上下文，再形态；没有跟随的信号不能当成高概率机会。PA Agent **不**做持仓后逐 K 管理。

---

## 1. 每根 K 线收盘后的固定流程（诊断/入场前）

1. 分类：趋势棒、十字星、内包、外包（细节 → 文件16）。
2. 角色：结构棒、信号棒、入场棒、确认棒。
3. 上下文：趋势 / 通道 / 区间 / 突破后 / 反转尝试。
4. 跟随：后续 1–2 根是否同向推进。
5. 更新**计划**（非持仓）：顺势、等二次入场、等突破测试、放弃、或 `order_type=不下单`。

---

## 2. 逐棒口诀

1. 先上下文，再形态。
2. 先信号质量，再入场计划。
3. 先二次入场，再评估反转诊断（逆势不下单）。
4. 先跟随，再确认可交易。
5. 先惯性，再逆势。
6. 看不懂，就等下一根 K 线。

---

## 3. Final Flag 逐棒检查

趋势运行较长时间后，若出现水平重叠整理（10–20+ 棒）：
1. 是否位于趋势末端、近 MM 或极值？→ 标 `final_flag`。
2. 顺趋势突破旗形是否 1–2 根无跟随？→ 失败 FF，勿追突破。
3. 是否出现反向信号棒 + 入场棒？→ 仅 `reversal_attempt` / `mtr` **诊断**（文件24、25）；**禁止逆势三价**
4. 整理区是否呈铁丝网？→ 文件21 优先，中部不交易。

---

## 4. 磁力位回测检查

- 价格是否回到**失败信号棒极点**或**失败入场价**？
- 是否接近 17t/41t 类陷阱区（文件22）？
- 写入 `watch_points` / `invalidation_condition`，**非**持仓平仓指令。

---

## 5. 2BR / 外包 / Micro Wedge

- **2BR**：双棒反转；须双确认才作 MTR 一部分（文件25）。
- **OD/OU**：铁丝网中不追；趋势线突破后可为入场棒（文件16）。
- **Micro Wedge（3–4 棒）**：标 `wedge` 时 bar_range 要短（文件14）。

---

## 6. High1/H2 与突破（指针）

- 计数与 H1/H2 规则 → **文件19**
- 突破失败 / 测试 → **文件18**
- 二次入场 → **文件15**
